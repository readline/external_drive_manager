CLI Commands Reference

📚 Quick Reference

Use the command line interface (CLI) to manage drives and search files. All operations are performed via ./cli.sh.

🗄️ Drive Management

Add a new drive

./cli.sh drive add <label> --serial <serial> --path <path> --depth <depth>

Example:

./cli.sh drive add 28T-02 --serial 00000000NT17P0Y8 --path /mnt/ext_2 --depth -1

List all drives

./cli.sh drive list
./cli.sh drive list --offline  # Show only unscanned drives

Scan a drive

./cli.sh drive scan <label>

Example:

./cli.sh drive scan 28T-02

Note: This creates a disk_scan.txt.gz file on the drive with the scan results.

Remove a drive

./cli.sh drive remove <label>

🔍 Search Files

Basic search

./cli.sh search <pattern>

Examples:

./cli.sh search "*.pdf"                    # All PDF files
./cli.sh search "report*.docx"             # Files starting with "report"
./cli.sh search "data???.txt"              # Use ? for single character
./cli.sh search "*backup*"                 # Files containing "backup"

Search with filters

./cli.sh search <pattern> [options]

Available options:

  • --drive <label> - Filter by drive
  • --ext <extension> - Filter by extension (e.g., .jpg)
  • --min-size <size> - Minimum size (supports K, M, G, T)
  • --max-size <size> - Maximum size (supports K, M, G, T)
  • --from-date <date> - From date (ISO format)
  • --to-date <date> - To date (ISO format)
  • --export <file> - Export to CSV

Examples:

./cli.sh search "*.mov" --drive 28T-02 --min-size 1G
./cli.sh search "*" --ext .jpg --max-size 10M
./cli.sh search "data*" --from-date 2024-01-01 --to-date 2024-12-31
./cli.sh search "*.csv" --export results.csv

📊 Other Commands

Get file details

./cli.sh file <file_id>

Show statistics

./cli.sh stats

💡 Tips

  • Search is case-sensitive and supports wildcards: * (any characters) and ? (single character)
  • Size units: K (KB), M (MB), G (GB), T (TB)
  • Search looks in filename, relative path, and full path
  • Scanning creates a disk_scan.txt.gz backup on the drive
  • Use the Browse and Search pages for interactive file exploration

🚀 Web Interface

The web interface provides interactive browsing and searching:

  • Dashboard - View statistics and recent drives
  • Browse - Browse files with filters
  • Search - Advanced search with filters

Note: Drive management (add/scan/remove) is only available via CLI.
